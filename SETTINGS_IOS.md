1. Открой Xcode:                                                                                                                                                         
   open ios/Runner.xcworkspace
2. В Project Navigator (левая панель) найди папку Runner
3. Правый клик на Runner → Add Files to "Runner"...
4. Выбери папку Settings.bundle (она внутри Runner/)
5. В диалоге обязательно:                                                                                                                                                
   - ☑️ Create folder references (НЕ "Create groups")                                                                                                                     
   - ☐ Copy items if needed — НЕ ставь галочку (файлы уже на месте)                                                                                                       
   - ☑️ Add to targets: Runner — обязательно выбрано
6. Нажми Add
7. Проверь: Settings.bundle должен появиться в Project Navigator с синей иконкой папки (folder reference)
8. Пересобери приложение и установи на устройство
